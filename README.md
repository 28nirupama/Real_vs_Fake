# Real_vs_Fake Text Detector

A simple web application and Python model to **detect whether a given text is written by a human or generated by AI**.  
Includes a FastAPI backend and a user-friendly frontend interface.

---

## Features

- Detects **Human vs AI generated text**.  
- Fun responses indicating detection results.  
- Combines **TF-IDF vectorization** with **extra text features**:
  - Emoji count  
  - Punctuation ratio  
  - Digit ratio  
  - Average word length  

---

## Technology

- **Backend:** Python, FastAPI  
- **Model:** LinearSVC (Support Vector Classifier)  
- **Frontend:** HTML, CSS, JavaScript  
- **Libraries:** scikit-learn, pandas, numpy, scipy, joblib

## Installation

1. Clone the repository:
```bash
git clone https://github.com/28nirupama/Real_vs_Fake.git
cd Real_vs_Fake

Create a virtual environment:

python -m venv venv
# Activate
# Windows: venv\Scripts\activate
# Linux/macOS: source venv/bin/activate


2.Install dependencies:

pip install -r requirements.txt


3.(Optional) Train the model:

python train_model.py

Running the Application

4.Start the FastAPI server:

uvicorn app:app --reload


5.Open your browser:

http://127.0.0.1:8000/


Enter text in the frontend and click Check!

Shows prediction: Human or AI
