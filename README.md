# Real_vs_Fake Text Detector

A simple web application that can **detect whether a given text is written by a human or generated by AI**.
It combines a Python-based FastAPI backend with a user-friendly frontend interface.

---

## Features

* Detects **Human vs AI-generated text**.
* Provides **fun, interactive responses** alongside predictions.
* Uses **TF-IDF vectorization** plus extra text features such as:

  * Emoji count
  * Punctuation ratio
  * Digit ratio
  * Average word length

---

## Technology Stack

* **Backend:** Python, FastAPI
* **Model:** LinearSVC (Support Vector Classifier)
* **Frontend:** HTML, CSS, JavaScript
* **Libraries:** scikit-learn, pandas, numpy, scipy, joblib

## Installation

1. **Clone the repository:**

```bash
git clone https://github.com/28nirupama/Real_vs_Fake.git
cd Real_vs_Fake
```

2. **Create and activate a virtual environment:**

```bash
python -m venv venv
# Windows:
venv\Scripts\activate
# macOS/Linux:
source venv/bin/activate
```

3. **Install dependencies:**

```bash
pip install -r requirements.txt
```

4. **(Optional) Train the model:**

```bash
python train_model.py
```

---

## Running the Application

1. **Start the FastAPI server:**

```bash
uvicorn app:app --reload
```

2. **Open your browser and visit:**

```
http://127.0.0.1:8000/
```

3. **Use the app:**

* Enter text in the frontend input box.
* Click **Detect!**
* The app will show whether the text is **Human or AI-generated** along with a **fun response**.
